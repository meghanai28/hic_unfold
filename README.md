# hic_unfold

Mechanism-structured conditional diffusion for deconvolving bulk Hi-C
into single-cell chromatin ensembles, using a loop-extrusion
configuration `z` (CTCF-anchored, oriented) as the latent that
conditions both the forward operator and the diffusion prior.

```
bulk Hi-C contact map
        │
        ▼  guided DDIM  +  per-cell z  +  soft-contact forward operator
        │
single-cell distance-matrix ensemble  →  3D coordinates (MDS)
```

## Headline results (chr21:28-30Mb, IMR-90, Bintu 2018 ground truth)

| Test | Pearson | 95% CI |
|---|---|---|
| Bulk fit vs measured contact map | **0.9871** | [0.9851, 0.9889] |
| 5-fold CV mean ± std (n_cells = 3,881) | **0.9874 ± 0.0003** | range [0.9869, 0.9878] |
| HIPPS-DIMES baseline (same data) | 0.9723 | [0.9686, 0.9755] |
| Naive (no z prior) ablation | 0.93 | drops 0.06 |

**Generalisation across loci and cell types (one fixed model, no retraining):**

| Dataset | n cells | Pearson | 95% CI |
|---|---|---|---|
| IMR90 chr21:28-30Mb (in-domain) | 3,881 | 0.9876 | [0.9858, 0.9890] |
| IMR90 chr21:18-20Mb (cross-locus) | 1,180 | 0.9733 | [0.9712, 0.9755] |
| K562 chr21:28-30Mb (cross-cell-type) | 2,817 | 0.9821 | [0.9798, 0.9840] |
| HCT116 chr21:28-30Mb untreated | 1,477 | 0.9868 | [0.9844, 0.9889] |
| K562 vs IMR90 control (cross-cell) | — | 0.9463 | [0.9395, 0.9522] |

Perturbation: attenuating the latent by `α = 0.5` reproduces the
measured HCT116 auxin cohesin-loss radius of gyration to within ~1 nm;
shuffled-z and random-mass controls miss it.

## Why this isn't another bulk-fit paper

The latent is biologically meaningful, not just a fitted bottleneck.
Five independent lines of evidence (see `outputs/47_proof_stack.png`):

1. **Track overlap** — encoder loop-propensity correlates with CTCF,
   RAD21, and ATAC bigWigs from ENCODE.
2. **Beats hard nulls** — circular-shift, block-shuffle, and AR(1)
   smooth nulls all give p < 0.001.
3. **Architectural > accessibility** — in a joint regression, CTCF +
   RAD21 explain more variance in propensity than ATAC.
4. **Perturbation specificity** — only the real (not shuffled, not
   random) latent recovers the auxin Rg shift.
5. **TAD recovery** — boundaries match the measured map at permutation
   p = 0.028.

Honest limitations are documented (single locus, marginal convergent-
CTCF trend p = 0.091, distance-matched loop-pair enrichment is null).

## Reproducing

```bash
# Python 3.11, torch + CUDA, scipy, scikit-learn, matplotlib, pyBigWig
pip install -r requirements.txt

# fetch raw data (~2 GB, one-time)
py -3 scripts/03_load_bintu.py
py -3 scripts/19_real_hic_deconvolution.py
py -3 scripts/20_ctcf_chipseq_overlay.py
py -3 scripts/35_rad21_chipseq_overlay.py
py -3 scripts/40_atac_overlay.py
py -3 scripts/44_convergent_ctcf.py

# run the full pipeline (skips work where checkpoints exist)
for i in 01 02 ... 49; do py -3 scripts/${i}_*.py; done
```

End-to-end wall time on a single RTX-class GPU: ~6 hours. Most scripts
re-use checkpoints from earlier steps, so partial reruns are cheap.

## Method, in one paragraph

A conditional v-prediction diffusion model is trained on simulated
single-cell distance matrices generated by a Gaussian-Rouse polymer
with loop springs determined by `z` (a loop-extrusion configuration
produced by a 1D LEF simulator with oriented CTCF barriers). At
inference, a sim-supervised encoder produces `z_hat` from each real
single-cell distance matrix; guided DDIM samples then push the
ensemble toward matching the observed bulk contact map under a
soft-contact forward operator `c = sigmoid((d0 - d) / tau)`. The
result is a population of single-cell distance matrices that (a)
average to the measured bulk, (b) carry loop-anchor structure inherited
from `z`, and (c) yield 3D coordinates via classical MDS.

Full design spec: see the architecture document referenced in
`hic_deconvolution_architecture.md` (private).

## Data sources

- **Bintu et al. 2018** — single-cell chromosome tracing of chr21
  (IMR90, K562, HCT116 ± auxin) at 30 kb.
- **Rao et al. 2014** — IMR-90 in situ Hi-C, GSE63525.
